# Outreach Draft Agents

These agents create Gmail drafts only. They must never send cold outreach automatically.

Every run should use the Supabase CRM, competitor research, price comparison data, market research, and automation outputs where available. The drafts should be ready for Daniel to review, edit, and send manually.

## Shared Rules

- Create 10 drafts per agent run when enough usable CRM leads exist.
- Use public, official, or CRM-stored contact emails only.
- Do not invent private contacts, job titles, recent events, relationships, budgets, or intent.
- Personalize from the CRM research notes, likely need, recommended offer, website, location, segment, and market-research context.
- Keep emails concise, human, respectful, and commercially useful.
- Include one clear reply CTA.
- Use the signature:

```text
Best,
Daniel
Fuze Ecoteer
```
